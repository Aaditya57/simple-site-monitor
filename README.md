# Simple Site Monitor

## Overview
This project is designed to monitor the uptime and performance of websites by checking their status at regular intervals. It alerts users when a site is down or experiences performance issues, ensuring they can respond quickly to maintain their online presence.

## Features
- **Uptime Monitoring:** Checks if a website is reachable.
- **Performance Metrics:** Measure response times and performance statistics.
- **Alerts:** Notifications via email or SMS when issues are detected.
- **Logs:** Keep historical data for analysis.

## Usage
Clone the repository and configure the site URLs to monitor. Set up your notification preferences to stay informed!
